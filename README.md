# ALURLRouter

简明扼要的描述功能


1. ALURLRouter可以得到任务处理进度，处理结果:成功/失败,成功信息/失败NSError
2. ALURLRouter有返回值
3. ALURLRouter参数更灵活,不仅限于url的query字符。而且可以是系统通用的类型如UIImage、NSDate等。
4. 可以实现灵活的权限控制
